# Reference Intake

Use this file to organize visual references before implementation.

## What to send

- Landing pages or websites you like
- Screenshots of sections you want to borrow ideas from
- Notes about what exactly you like in each reference
- Notes about what you do not want repeated

## How references will be used

- Structure: section order, pacing, density
- Visual language: typography rhythm, image treatment, card style, whitespace
- Conversion patterns: CTA placement, proof blocks, offer framing
- Motion: reveal style, scrolling behavior, interaction feel

## Reference log

Paste each reference in this format:

```md
Reference:
Link or file:
What I like:
What I do not want to copy literally:
Priority: high | medium | low
```

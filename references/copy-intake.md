# Copy Intake

Use this file when defining the content and conversion logic of the page.

## What to send

- Product or service name
- What the offer is
- Who it is for
- Main promise or transformation
- Main CTA
- Secondary CTA if any
- Core objections
- Proof: testimonials, numbers, clients, cases
- FAQ items
- Any required sections

## Recommended format

```md
Product:
Audience:
Offer:
Primary CTA:
Secondary CTA:

Main promise:

Pain points:
- 

Benefits:
- 

Proof:
- 

Objections:
- 

FAQ:
- Q:
  A:
```

## Notes

- If the copy is raw, it can still be sent this way and refined later.
- If the user sends full sales copy, preserve the meaning and improve hierarchy instead of rewriting from scratch by default.
